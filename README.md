# Astronauta Varado

Juego en Unity 6 sobre un astronauta varado en un planeta extraño.
El jugador debe salir de la nave, encontrar una batería en lo alto de una montaña
con ayuda de un perrito espacial, sortear plataformas de salto, y volver a la
nave para reestablecer los sistemas.

## Tecnologías
- Unity 6 (6000.x)
- URP (Universal Render Pipeline)
- Input System (nuevo)
- Cinemachine 3.x
- WebGL

## Cómo jugar
WASD: caminar
Mouse: mirar
Shift: correr
Espacio: saltar
E: interactuar

## Compilación
Plataforma destino: WebGL.
